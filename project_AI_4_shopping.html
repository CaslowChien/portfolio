Real-Time Recommendation System for Online Shopping



During my internship, I developed a real-time recommendation system for online grocery shopping with a small team over three months. I achieved a search time of 0.05 milliseconds and a recall rate of 97% through rigorous stress testing. I optimized nearest neighbor search methods for the company’s hardware and utilized Microsoft SQL Server to retrieve product data, constructing a DNN model for recommendations. By refining nested code, I reduced processing time by 50% and designed data batch formats to resolve conflicts, enhancing overall performance.

  • Developed a recommendation system for online grocery shopping in 3 months with a small team of 2.
  • Conducted stress tests to reduce the search time to 0.05 milliseconds with a recall of 97%.
  • Retrieved product data with Microsoft SQL Server, reduced the processing time by 50% with modify the nested code.
  • Designed data batch formats to resolve data conflicts.
